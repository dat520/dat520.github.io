# DAT520 Distributed Systems

> Course information and resources for DAT520 Distributed Systems at University of Stavanger

## Welcome to DAT520

This course provides a comprehensive introduction to distributed systems concepts, algorithms, and practical implementation.
You'll learn about consensus protocols, replication, fault tolerance, and building reliable distributed applications.

## Course Overview

- **Distributed Systems Fundamentals**: Understanding the challenges of distributed computing
- **Consensus Protocols**: Paxos, Raft, and quorum-based systems
- **Fault Tolerance**: Building reliable systems that handle failures
- **Replication**: Data consistency and synchronization
- **Practical Implementation**: Building distributed systems in Go
